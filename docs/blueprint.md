# **App Name**: CryptoAutomat Algo Trading Platform

## Core Features:

- Dashboard Layout: Set up a professional dashboard layout with a sidebar and main content area using Next.js, Tailwind CSS, and ShadCN UI components.
- Visual Strategy Editor: Implement a visual strategy editor using @xyflow/react, allowing users to create trading strategies visually.
- CCXT Exchange Integration: Integrate CCXT for connecting to various cryptocurrency exchanges, handling real-time data and order execution.
- Algorithmic Trading Bot: Develop a trading bot that executes strategies based on technical indicators and user-defined rules.
- Technical Indicators: Incorporate technicalindicators for financial analysis to support strategy development.
- Strategy Compiler: Create a utility to compile visual strategies into JSON format for execution by the trading bot.
- Cron Job for Bot Execution: Implement a cron job to trigger the trading bot on a scheduled basis, enabling automated trading.

## Style Guidelines:

- Background color: Dark gray (#1E1E1E) to provide a professional, dark mode experience.
- Primary color: Vibrant purple (#A78BFA) to highlight key elements and create a modern feel.
- Accent color: Electric blue (#7DD3FC) for interactive components and important actions.
- Body font: 'Inter' sans-serif for a clean, modern and readable interface.
- Headline font: 'Space Grotesk' sans-serif for a techy, scientific feel in headers.
- Use 'lucide-react' icons for clear and consistent visual representation of features and actions.
- Implement a responsive layout with a sidebar navigation and a main content area for a seamless user experience on different devices.
- Subtle animations for interactive elements, such as button hover effects and transition animations, to enhance user engagement.
